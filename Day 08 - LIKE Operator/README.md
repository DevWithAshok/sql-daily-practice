# Day 08 - LIKE Operator in SQL

## Topic

Using LIKE operator for pattern matching

## Explanation

The LIKE operator is used to search for a specified pattern in a column.

* `%` represents zero or multiple characters
* This query returns products whose names start with **Bourbon**

## Example Matches

* Bourbon Biscuit
* Bourbon Chocolate
* Bourbon Cream
